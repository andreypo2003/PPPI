Проект «Блокнот»

О проекте

«Блокнот» – проект, демонстрирующий базовые возможности текстового редактора, включая работу с файлами, проверку орфографии, смену тем оформления, автосохранение и т. д.

Структура репозитория

В папке docs хранятся лабораторные работы и файлы для сайта GitHub Pages. Вы можете найти:

  • lab2.cs – основной кода.
  • README.md – общее описание.
  • PPPI4.xml – файл документации.
  • PPPI5.hta – HTA-руководство.
